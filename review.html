Sandbox review flow

This page summarizes the exact end-to-end flow demonstrated for TikTok app review.

Products demonstrated

  • Login Kit
  • Content Posting API

Scopes demonstrated

  • user.info.basic
  • video.upload

Review environment

  • Platform: Web
  • Website URL: https://willyhub31.github.io/tiktokpolicies/
  • Redirect URI: https://willyhub31.github.io/tiktokpolicies/callback/
  • TikTok review demonstration: sandbox test account

What the demo video shows

  1. Open the public Le8Geek website.
  2. Start TikTok Login Kit on the web app.
  3. Authorize the sandbox TikTok account.
  4. Return to the registered callback URL on the same website.
  5. Select an approved video in the Le8Geek workflow.
  6. Upload the selected file with Content Posting API.
  7. Show the uploaded draft/private result in the sandbox TikTok account.

Why sandbox is used

Le8Geek follows TikTok review guidance and uses sandbox to demonstrate the integration before production approval. The production app only requests the products and scopes that are shown in the sandbox demo.
