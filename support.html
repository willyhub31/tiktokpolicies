Le8Geek support covers web sign-in, callback handling, and TikTok draft upload workflow issues.

Email support

le8geek@gmail.com

Response window

Within 2 business days for standard support requests.

Why are uploads private or draft-first?

This lets the user review caption text, disclosure, and affiliate links inside TikTok before final publication.

Does Le8Geek publish without permission?

No. Every upload requires explicit TikTok authorization and a user-triggered action inside the workflow.

How can a user remove access?

The user can revoke access from the TikTok connected apps settings at any time.

Which TikTok products are used?

Login Kit and Content Posting API only. Share Kit is not used by the Le8Geek web app.
