TikTok Creator Tools

Generate Product Videos, Review Privately, Then Publish to TikTok

Le8Geek Creator Suite is a creator workflow app for e-commerce affiliates. It generates product media, stores campaign assets, and uploads TikTok drafts for manual review before final publication.

Open Product Docs Contact Support

Core Features

  • Reference-based image and video generation for product campaigns.
  • Airtable-driven workflow for prompt, asset, and publishing status tracking.
  • TikTok Login Kit + Content Posting API integration.
  • Private-first posting flow for editing captions and affiliate links before publishing.

Authorized TikTok Usage

  • Login Kit for user authentication and account linking.
  • Scope: user.info.basic for identity and account mapping.
  • Scope: video.upload for draft upload workflow.
  • Users explicitly authorize before any upload action.

End-to-End Workflow

  1. User logs in with TikTok via OAuth.
  2. User selects product references and campaign settings.
  3. System generates media drafts and stores them in Airtable.
  4. System uploads selected video to TikTok as a draft/private flow.
  5. User reviews and edits post details (including affiliate links) before final publish.
