Product Documentation

This page documents how Le8Geek Creator Suite integrates with TikTok for creator publishing workflows.

Supported Products and Scopes

  • Login Kit
  • Content Posting API
  • Share Kit (optional usage)

Current scopes: user.info.basic, video.upload

How Upload Works

  1. User signs in with TikTok OAuth.
  2. User selects a generated campaign video from Airtable workflow.
  3. System uploads video draft through TikTok API.
  4. User reviews post content, updates copy and affiliate links, then publishes.

Operational Notes

  • The app uses explicit user authorization and does not post to accounts without consent.
  • Draft/private-first flow is used for quality checks and policy-safe review.
  • Users remain responsible for final content compliance and disclosure obligations.
