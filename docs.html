Integration documentation

This page documents the exact public web setup used by Le8Geek for TikTok Login Kit and Content Posting API.

Official website

https://willyhub31.github.io/tiktokpolicies/

Redirect URI for Web

https://willyhub31.github.io/tiktokpolicies/callback/

Selected TikTok products and scopes

Product Scope Purpose in Le8Geek
Login Kit user.info.basic Authenticate the user and associate the authorized TikTok account with the upload workflow.
Content Posting API video.upload Upload an approved video from the Le8Geek workflow as a draft or private post.
Share Kit is not used. Le8Geek is a web app and only requests the products shown above.

Web upload flow

  1. User opens the Le8Geek web app and starts TikTok sign-in.
  2. User authorizes the app through Login Kit.
  3. TikTok redirects the user back to the registered callback URL on this website.
  4. User chooses an approved product video that was prepared in the Le8Geek workflow.
  5. Le8Geek uploads the selected video through Content Posting API.
  6. User completes final caption edits, disclosure, and affiliate links inside TikTok.

Authorization model

Le8Geek only acts after the user explicitly authorizes the account. The app does not post to TikTok without a user-triggered workflow.

Review environment

TikTok review demonstrations are performed in sandbox mode with the same public website and redirect path used by the web app.
